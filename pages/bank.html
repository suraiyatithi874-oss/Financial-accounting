4. Bank Reconciliation

Purpose

The Bank Reconciliation module allows you to match and reconcile bank transactions recorded in the system against actual bank statements. This process ensures financial accuracy, helps spot discrepancies, and maintains clean audit trails for organizational transparency.

Navigation

Menu Path: Main Menu → Bank Reconciliation Entry

üïπÔ∏è How to Use

Step 1: Apply System Filters

  • Select the relevant Region / Area / Branch.
  • Define the Voucher Date Range to pull bank-related transactions.
  • Click the Search button to display all entries within the time frame.

Step 2: Reconcile Entries

  • Review the transactions populated in the grid.
  • Locate the Reconciliation Date input field for each entry.
  • Input the date the transaction appeared on your physical bank statement.
  • Click Save once all dates are confirmed.
