2. Chart of Accounts (COA)

Purpose

This module allows users to define and manage the full structure of accounts used in their organization's financial system. It organizes financial accounts under categories like Assets, Liabilities, Income, and Expenses using a hierarchical approach.

Navigation

Menu Path: Main Menu → Chart of Account (COA)

Operations

The COA List Page provides the following tools to manage your financial structure effectively:

List Page Control

Displays the Account Head List with options for Ledgers, Search, Publish/Unpublish, Export, and Print.

Branch & Account Type Filters

Allows filtering of the list based on specific Branch or Account Category (Assets, Liabilities, etc.).

Search Bar

Quickly locate accounts using keywords to navigate large lists instantly.

Account Head List

Displays the Name, unique Code, and Type of each account charted in the system.

Expandable Hierarchy

Click folder icons to expand or collapse account groups to view parent-child relationships.

2.2 Chart Of Account List Entry

Form Purpose

This form allows you to add a new ledger or account head into the system. Each field defines how the account will behave in transactions and reports.

üìã Field Descriptions

Field Name Requirement & Description
Ledger Name [Required] Type the account name (e.g., Cash, Sales Revenue).
Account Code [Required] Input a unique numerical code (e.g., 101).
Parent Account Choose Grand Parent for top-level categories or select a specific parent for sub-accounts.
Account Type [Required] Choose from: Assets, Liabilities, Equity, Income, or Expenses.
Ordering After Set the visual position under the parent (e.g., At First, or after another account).
Assign Project Tick "All Project" for global availability or "General" for common system use.
Description Optional field for purpose notes, branch addresses, or additional details.
Is Group Head Checked: Account acts as a folder for sub-accounts.
Unchecked: Account acts as a "Leaf" (Transactional) ledger.

Step 9: Save Button
